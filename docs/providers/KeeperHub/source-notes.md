## Project-useful facts

- KeeperHub positions itself as execution and reliability infrastructure for onchain agents.
- Workflows combine triggers, actions, conditions, and data references between nodes.
- API access supports programmatic workflow and execution management.
- AI tool surfaces include MCP and agentic wallet docs.
- The public repository is TypeScript and includes an app, plugins, execution tracking, and workflow APIs.

## Claims to verify during implementation
