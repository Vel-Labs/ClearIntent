Sources: [Trust Wallet GitHub organization](https://github.com/trustwallet), [WalletConnect GitHub organization](https://github.com/WalletConnect).

| Repository | Surface | License observed | ClearIntent relevance |

| [trustwallet/wallet-core](https://github.com/trustwallet/wallet-core) | Cross-chain wallet core | Apache-2.0 | Low-level wallet implementation reference. |

## ClearIntent implication

Trust Wallet should be treated as a mobile WalletConnect candidate until an exact EIP-712 signing and preview path is tested.
