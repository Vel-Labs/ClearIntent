# EIP-712

Current level: L0 researched only.

Target level: L2 EIP-712 signable for the first implemented wallet path.

EIP-712 is ClearIntent's baseline signing primitive for typed agent intents. It gives wallets a structured payload with a domain, primary type, typed fields, and message values.

ClearIntent relevance:

- `AgentIntent` should be transformed into deterministic EIP-712 typed data.
- The domain must bind chain, verifier, and version where applicable.
- The message must include authority-critical fields such as policy hash, nonce, deadline, executor, bounds, and audit references.
- EIP-712 support does not automatically guarantee high-quality wallet display.
