## Project-useful facts

- 0G documentation exposes a machine-readable docs index and full text surface for AI/RAG workflows.
- Storage has TypeScript and Go options, CLI workflows, file/directory upload and download, KV operations, and indexer-mediated upload paths.
- Compute exposes account/provider workflows and OpenAI-compatible inference patterns.
- INFT / ERC-7857 is documented but should remain stretch scope for ClearIntent.
