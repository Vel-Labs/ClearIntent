# Repo Truth

This folder holds the project’s durable source-of-truth material.

## Contents

```text
THC_METHODOLOGY.md          general Truth, Hardening, Clarity methodology
THC_IN_THIS_REPO.md         ClearIntent-specific mapping
OPERATING_DOCTRINE.md       project doctrine and non-negotiables
TRUTH_BOUNDARIES.md         where truth lives and where it does not
```

## Rule

Repo truth must be inspectable. It should not live in agent memory, private chat, or assumptions that new contributors cannot verify.
